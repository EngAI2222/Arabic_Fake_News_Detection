# Arabic_Fake_News_Detection
يهدف هذا المشروع إلى تطوير نظام ذكي يعتمد على معالجة اللغات الطبيعية (NLP) وتقنيات التعلم العميق (Deep Learning) للكشف عن الأخبار المزيفة باللغة العربية. يقوم النظام بتحليل النصوص الإخبارية ومعالجتها باستخدام عدة مراحل مثل تنظيف النصوص وتقسيمها وتحويلها إلى تمثيل رقمي، ثم يتم تدريب نموذج LSTM لتصنيف الأخبار إلى أخبار حقيقية أو مزيفة
